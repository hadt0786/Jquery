Jquery Selectors

The element Slector

$(document).ready(function(){ $("button").click(function(){ $("p").hide(); }) });


The #id Slector

$(document).ready(function(){ $("button").click(function(){ $("#demo").hide(); }) });

The class Slector

$(document).ready(function(){ $("button").click(function(){ $(".demo").hide(); }) });

The * Slector

$(document).ready(function(){ $("button").click(function(){ $("*").hide(); }) });
